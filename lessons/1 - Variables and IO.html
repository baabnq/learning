Lesson 1 - Variables and IO

Variables

put _a = 100;
put _b = 200;
put _a = _b;

Variables in Baabnq are always prefixed by an underscore "_" and are always assigned with the "put" command. However, they are not declared but inferred by the compiler. Meaning, a variable is created when it's first assigned to something. Baabnq is a typeless language, which means that variables are not created with a type and are always 16-bit ( 0 - 65535 ) unsigned integers.

Output

Print

print 100;
put _a = 200;
print _a;

The "print" command takes a variable or a constant and outputs it to stdout as an unsigned decimal integer. So as a result, the program above will print the following to the stdout: 

100
200

Putchr

putchr 10;

The "putchr" command also takes a variable or a constant, but it outputs the corresponding unicode character. So the above program would output a line feed.

Input

input _a;
input _b;

The "input" command reads an unsigned decimal integer from stdin and writes it to the specified variable.
