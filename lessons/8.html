A Routine Signature is a comment the preceeds the actual routine and specifies the routine's type. It is comprised of two parts: The type signature and the implementation signature.


Type Signature

The type signature usually comes first in the routine signature and details the in- and out-parameter names along with their types in calling order.
(InParam::InType, ...) => (OutParam::OutType, ...)
